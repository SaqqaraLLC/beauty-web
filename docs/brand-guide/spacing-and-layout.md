
Saqqara LLC – Brand & Spacing Guide (v1.0)

This guide locks spacing, layout, and logo usage rules.

Core spacing unit: 8px
Do not use arbitrary spacing values.

Logo safe zone: height of lowercase 'a' around logo.
Sections: minimum 96px vertical spacing.
Cards: 32px gap, 24–32px internal padding.
Buttons: 16–24px horizontal padding, 12–16px vertical.

Principle:
If unsure, increase spacing. Never reduce spacing to fit content.
